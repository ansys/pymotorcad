Geometry fitting
================
Geometry fitting functions are used to find line and arc
entities to fit a list of coordinates within a defined
tolerance for Motor-CAD Adaptive Templates geometry using
PyMotorCAD.
More information on Adaptive Templates is available
in the :ref:`ref_user_guide` under :ref:`ref_adaptive_templates_UG`.
